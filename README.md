# 🎨 Python Graphics & Computer Vision Projects

A collection of beginner-friendly Python projects demonstrating **Computer Vision**, **Image Processing**, and **Graphics Programming** using **OpenCV**, **NumPy**, and **Matplotlib**.

This repository includes:

- 🇮🇳 Indian Flag Drawing
- 😊 Face & Eye Detection using Haar Cascade
- ♟ Chessboard Generation

---

# 📌 Project Overview

This repository contains three Python projects designed to help beginners understand graphical programming and computer vision concepts.

### 🇮🇳 Indian Flag

A Python program that draws the Indian National Flag using Matplotlib.

### 😊 Face Detection

A Computer Vision project that detects human faces and eyes using OpenCV's Haar Cascade Classifier.

### ♟ Chessboard

A Python graphics project that generates an 8×8 chessboard pattern using NumPy and Matplotlib.
