# Image Processing using Python

A collection of Image Processing projects implemented in Python using OpenCV, NumPy, and Matplotlib. This repository demonstrates basic image manipulation, image cleaning, object detection using Haar Cascade, pattern generation, and image creation.

---

## Project Overview

This repository contains Jupyter notebooks for learning and practicing Image Processing concepts.

### Technologies Used

- Python 3
- OpenCV
- NumPy
- Matplotlib
- Google Colab / Jupyter Notebook

---

## Notebooks

### 1. Image Cleaning & Manipulation

**File:** `Image_cleaning_manipulation.ipynb`

### Features

- Read image
- Resize image
- Crop image
- Rotate image
- Flip image
- Blur image
- Convert to Grayscale
- Save processed image

---

### 2. Haar Cascade Detection

**File:** `cascade.ipynb`

### Features

- Load Haar Cascade classifier
- Detect faces
- Draw bounding boxes
- Display detected faces
- Real-time detection support (optional)

---

### 3. Chess Pattern Generation

**File:** `chess.ipynb`

### Features

- Create chessboard pattern
- NumPy matrix operations
- Alternate black and white squares
- Display using Matplotlib

---

### 4. Indian Flag Generation

**File:** `flag.ipynb`

### Features

- Draw Indian Flag
- Saffron, White and Green stripes
- Blue Ashoka Chakra
- Adjustable width and height
- Display using Matplotlib

## Installation

Clone the repository

```bash
git clone https://github.com/dhrumilkhatri1-commits/image-processing-.git
```

Go to project folder

```bash
cd image-processing-
```

Install required libraries

```bash
pip install opencv-python numpy matplotlib
```
